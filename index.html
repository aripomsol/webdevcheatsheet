Basic CSS Cheatsheet

Selectors

Tag Name Description
< * > The Universal Selector Type selector to select all items in an HTML
< div > Division Used as a container for HTML elements
< class > Class Global attribute that allows CSS and JavaScript to select and access specific HTML elements

Font Properties

Tag Name Description
font-family: Font Name Indicates the fonts to be displayed
font-size: Font Size Indicates the font size to be displayed
font-family: Font Style Indicates the fonts to be displayed
text-align: Text Align Designates where the HTML elements are displayed on the page (left, right, center, justified)
